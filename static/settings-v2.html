Setup Guide

Choose one of the two methods below.

Option A — Local CLI Auth (server has a browser)

1. Install Claude Code CLI

curl -fsSL https://claude.ai/install.sh | bash

Then restart your shell or open a new Terminal tab.

2. Log in

claude

On first run it opens a browser login flow. Complete the login, then close the CLI.

3. Set Auth Mode → Local (CLI login) above, then test with a new Claude worker tab.

Option B — OAuth Token (headless server or remote setup)

Run these on any machine with a browser (your laptop, desktop, etc.):

1. Install Claude Code CLI (if not already installed)

curl -fsSL https://claude.ai/install.sh | bash

2. Generate a token

claude setup-token

This opens a browser login flow and saves the token locally.

3. Copy the token

cat ~/.claude/.credentials.json | python3 -c "import json,sys; print(json.load(sys.stdin)['claudeAiOauth']['accessToken'])"

Copy the sk-ant-oat01-... token that prints out.

4. Paste in KukuiBot

In the Credentials section above:

  • Paste the token into the OAuth Token field and click Save
  • Set Auth Mode to Configured (key/token)

5. Test it

Create a Claude Code worker tab and send a quick prompt to confirm it's connected.
