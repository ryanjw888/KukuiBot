Installation Instructions
🍎 macOS
  1. Open the downloaded .pem file
  2. Keychain Access opens automatically
  3. Double-click the certificate (named "mkcert …")
  4. Expand Trust → set to Always Trust
  5. Close and enter your password to confirm
📱 iPhone / iPad
  1. Tap the Download button above (in Safari)
  2. Tap Allow when prompted to download the profile
  3. Open Settings → a "Profile Downloaded" banner appears at the top → tap it
  4. Tap Install → enter your passcode → tap Install again
  5. Go to Settings → General → About → Certificate Trust Settings
  6. Toggle ON for the mkcert certificate
🪟 Windows
  1. Double-click the downloaded file
  2. Click Install Certificate
  3. Select Local Machine → Next
  4. Choose Place all certificates in the following store
  5. Browse → select Trusted Root Certification Authorities
  6. Click Finish
🐧 Linux
  1. Copy the file to /usr/local/share/ca-certificates/kukuibot.crt
  2. Run sudo update-ca-certificates
