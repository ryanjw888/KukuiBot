The Verdict
These aren't competitors — they're different answers to different questions.

KukuiBot

A batteries-included AI workstation. Multi-provider chat, browser automation, Gmail integration, network auditing, worker delegation, cron scheduling, and a full responsive web UI — all in one self-hosted package. It's the "I want one tool that does everything" approach. The tradeoff is complexity: 59K lines of code, 80+ source files, and a larger surface area to maintain.

Best for: Power users, home lab operators, teams who want a private AI command center with deep system integration and don't mind a larger codebase.

Claude Telegram Relay

A beautifully minimal relay — 3 source files, 3 npm packages, and one clear job: pipe Telegram messages through Claude Code and back. The semantic memory via Supabase and proactive check-ins give it a personal-assistant feel that goes beyond simple chat. The tradeoff is scope: no web UI, no multi-provider support, and heavy reliance on external services (Supabase, Telegram, OpenAI).

Best for: Individuals who want a personal AI companion in Telegram with smart memory, value simplicity and auditability, and are comfortable with cloud dependencies.
